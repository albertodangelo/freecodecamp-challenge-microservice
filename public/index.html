API Basejump - Timestamp microservice

Als Parameter kann ein Datum oder ein Unix-Timestamp oder ein Datum im englischen Format übergeben werden. Es wird immer der Timestamp und das entsprechende Datum im JSON-Format (deutsch) zurückgegeben. Bei einer fehlerhaften Eingabe wird "null" als Wert geliefert.

Beispiel der Eingabe

https://timestamp-ms.herokuapp.com/December%2015,%202015
https://timestamp-ms.herokuapp.com/1450137600

Beispiel der Ausgabe

{ "unix": 1450137600, "natural": "15. Dezember 2015" }
